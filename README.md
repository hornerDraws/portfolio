# Portfolio Website Setup

## Overview
This repository contains the code for my portfolio website. The website showcases my projects, skills, and experiences. In this README, I will guide you through the setup process and how to add images to the portfolio.

## Setup Instructions
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/hornerDraws/portfolio.git
   cd portfolio
   ```

2. **Install Dependencies**:
   Ensure you have the necessary packages installed. Run:
   ```bash
   npm install
   ```

3. **Run the Development Server**:
   Start the development server to view your portfolio live:
   ```bash
   npm start
   ```
   Open your browser and navigate to `http://localhost:3000`.

## Adding Images
To add images to your portfolio:
1. Place your images in the `assets/images` directory.
2. Update the corresponding HTML or Markdown files to reference the images. Use the following syntax:
   ```html
   <img src="assets/images/your-image.jpg" alt="Description of image">
   ```

## Conclusion
Feel free to customize the content and style of the portfolio to suit your personality. Happy coding!
